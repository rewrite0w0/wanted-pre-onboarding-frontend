# wanted-pre-onboarding-frontend

## 프로젝트의 실행 방법

1. 레포지토리 클론

```bash
git clone https://github.com/rewrite0w0/wanted-pre-onboarding-frontend.git
```

2. 패키지 설치 및 실행

```npm
npm install

npm start
```

3. 조작
   [localhost:3000](http://localhost:3000)
